## Vieningos retransliatorių pavadinimų sistemos privalumai

1. Paprasta nustatyti retransliatoriaus vietą pagal jo pavadinimą
2. Paprasčiau sudaryti žinutės pristatymo kelią
3. Galima suprasti žinutės gavimo kelią nenaudojant žemėlapio
4. Lengviau naudotis paieška ir filtruoti kontaktus
5. Pavadinimų sistema sudaro rimto tinklo įspūdį
6. Vieninga pavadinimų sistema siunčia pozityvią žinutę apie vieningą MeshCore bendruomenę

## Formatas

Retransliatoriaus pavadinimą rekomenduojama formuoti pagal šią struktūrą:

```
{Kodas}-{Mikrorajonas|Vietovė}-{ID}[-Kryptis]
```

- `Kodas` - miesto kodas, jeigu miesto ribose, apskrities kodas, jeigu už miesto ribų
- `Mikrorajonas` -  miesto mikrorajonas, miesto gyvenvietė, miesto žymi vieta
- `Vietovė` - apskrities miestas (išskyrus apsk. centrą), miestelis, gyvenvietė, kaimas, žymi vieta
- `ID` -  retransliatoriaus viešojo rakto pirmieji 2 simboliai
- `Kryptis` (neprivaloma) - retransliatoriaus spinduliavimo kryptis: N,NE,E,SE,S,SW,W,NW
